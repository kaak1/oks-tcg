# 图片替换说明

当前版本使用原创 SVG 占位图，未使用官方 Pokémon Logo、官方角色插画或网络抓取素材。

以后替换真实图片时，请保持路径不变，或同步修改 `data/products.ts` 与 `data/categories.ts`：

- `public/images/products/`：商品图、Hero 盲包、飞出卡牌占位图
- `public/images/categories/`：热门分类图片
- `public/images/events/`：实体市集活动图片
- `public/images/og.svg`：社交分享预览图
- `public/favicon.svg`：网站图标

建议真实商品图尺寸：

- 商品图：至少 1040 x 960，主体居中，深色或透明背景更适合当前视觉。
- 分类图：至少 920 x 780。
- 市集图：至少 1040 x 904。

替换为真实产品照片前，请确认你拥有图片使用权，并避免使用未经授权的官方角色插画作为页面装饰。
